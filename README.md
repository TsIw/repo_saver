# R.E.P.O. セーブデータマネージャ

R.E.P.O. セーブデータのバックアップを自動で行うツールです。  

## 機能概要
- セーブデータ更新を検知して自動バックアップ
- 手動操作によるバックアップ／リストア／バックアップ削除
- バックアップに対するメモ記録
- タスクトレイ常駐

## インストール方法
GitHubのReleasesから最新のインストーラをダウンロードしてインストールしてください。  
※インストール先のフォルダにバックアップファイルや設定ファイルを保存するため、管理者権限が必要となる「C:¥Program Files」などへはインストールしないでください。

## アンインストール方法
Windowsの「設定 > アプリ」から「RepoSaver」をアンインストールしてください。

## 開発関連情報
- 開発環境： Tauri + Rust + Vue.js
- 使用したツール： Google Antigravity

※本ツールは生成AI技術を用いて開発されており、動作上の不安定などがある可能性があります。
